Which are the technologies that OneDB is using?

OneDB is built only with free technologies. Practically, it is a beautifull thing made up from the ground with this thing in mind

Take a look at the list of requirements that you should meet in order to install OneDB:

  • Linux operating system (recommended, but also works on Windows too). A recommended Linux distribution would be a debian based one.
  • Apache WebServer
  • PHP 5.3.6
  • mongoDB (recommended the latest version)
  • ffmpeg (for video transcoding on the fly)
  • Image editors are provided by the guys from Pixlr.com website, which built a wonderful tool for image editing
  • jsPlatform - which is the framework web operating system used to build OneDB backend, which at by itself it requires:
    • Google Chrome or Safari web browser (alhtough might work on Firefox, but not tested yet)
    • MySQL database system
    • Rich interface controls were integrated into jsPlatform from the following projects:
      • TinyMCE editor
      • Ace Ajax.org Cloud 9 Editor
      • Valums file uploader

These technologies, are the main puzzle components that were combined for building OneDB

All competitors are requiring you to install and use specific versions of these products, but OneDB philosophy is to encourage users to install the latest versions of it's software components
