OneDB -> Connect to server

Clicking this menu item, a new window will apear:

From this dialog you can start a new session on your OneDB database. You can connect to the database in two different ways.

Either provide a database Hostname, Database name, User Name and a Password, either by using an allready connection saved in the past from the dropdown below, and click on the button Load. If you Load a connection, all fields will be auto completed with the values you typed in last time you connected and saved that connection.

So, basically, this is the flow:

  • First time connection to the site:
    Type in the Host, Database Name, UserName and Password provided by your site administrator or developer.
    If you want that information to be auto completed next time you will open OneDB, also check the Remember Connection checkbox.
    Do not forget to save your connection by clicking the Save button, which will put the connection into the connection manager dropdown.
    A good practice when saving the connection, would be to save the connection without the password. Passwords should only be saved "in your mind", in order to avoid intruders into systems.
  • To use a previous saved connection:
    1. Select the previous saved connection from the dropdown
    2. Click the button Load 3. Click the button Connect
