OneDB interface description

Interface Components

OneDB interface is divided into 4 main components:
1. Menu
2. Toolbar
3. Tree Panel
4. View Panel

1. Menu

From the window menu, you can do most of the operations in OneDB, although some menu items have keyboard shortcuts assigned, in order to speed the things up

We'll speak about all entries in the menu, and what are their functions

At this point, the menu is composed by these elements:

  • OneDB - used for mongoDB database connections and database related stuff
  • File - used for articles / files / categories management (creation, upload, etc.)
  • Edit - used for renaming, cutting, pasting and deleting items
  • View - used for customising the look and feeel of the interface
  • Tools - used mostly by programmers, for managing sub-components of OneDB
  • Help - used for obtaining help about the application

2. Toolbar

From the toolbar you can do at any time most application common operations, such as refreshing content, modify the look of the view panel, etc.

You can safely play with the toolbar buttons, as there is nothing you can do to mess the thins up

3. Tree Panel

The tree panel is a point from where you can easily navigate into data locations, by simply clicking the items of the tree.

Also, the tree has a drag and drop feature, from where you can arrange categories in database, meaning that the order that you see in the backend, that order will be used by programmers when they are listing stuff on your website.

Category folders displayed on the tree have some meaning:

  • - Undiscovered Offline folder. The folder is offline, and OneDB has not yet navigated yet inside that folder, meaning that it don't know it's sub-folders structure.
  • -Undiscovered Online folder. The folder is online, and OneDB has not yet navigated inside that folder, meaning that it don't know it's sub-folders structure.

4. View Panel

The view panel is the region where are displayed items from the focused element from the tree panel. This means than whenever you click a folder in the tree panel, it's content is loaded displayed in the view panel.
